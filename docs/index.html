詳しくない人が、放っておいても失わないための NAS です。

01 — 安いディスクで組める

いちばん安いディスクを活かす

既成の商用NASは安いディスクを受け付けないことが多い。KCI-NASは中古PCと最安ディスクで組め、SSDキャッシュで速度を補います。

02 — 開かない前提

管理画面を開かなくても動く

高機能なNASほど使いこなすのに知識が要る。KCI-NASは、一度も管理画面を開かなくても勝手に守るよう作っています。

03 — 知らせる・守る・戻す

保存だけで終わらせない

多くのNASは「保存」が中心。KCI-NASは保存に加えて、異常を早めに知らせ、守り、パソコンが壊れても戻すことに重点を置きます。

!

速さを競う製品ではありません。 速さは回線とディスクで決まります。KCI-NASが目指すのは「当たり前の安心」を、いちばん手の届く価格で成立させることです。

!

バックアップが前提です。 安いディスクを安心して使えるよう遠隔ミラーや丸ごとバックアップを用意しています。大切なデータは必ず複数の場所へ。

はじめる

電源を入れて、あとは忘れていい。

Ubuntu Server のターミナルに、下の1行を貼るだけ。あとは http://<NASのIP>:8080 を開き、admin / admin でログイン（すぐにパスワードを変更）すれば、見守りが始まります。

# Ubuntu Server のターミナルに、この1行を貼って Enter
$ curl -sL https://github.com/KimYuki3/KCI-NAS2/releases/latest/download/install.sh | sudo bash
